Mapbox Navigation — Which approach?

Pick the one that matches what you're going for

A

Turn-by-turn UI (Directions API)

Build a navigation overlay using the step data already being fetched from Mapbox Directions. Shows current maneuver, distance to next turn, step list, ETA. Voice guidance via browser TTS. Flood-aware routing stays intact.

Pros

  • No new SDK or package
  • Full control over UI
  • Flood avoidance works as-is
  • Free tier friendly

Cons

  • Must build step-tracking logic
  • No lane guidance
B

Mapbox Navigation SDK (mapbox-navigation-js)

Install the official Mapbox Navigation SDK. Embeds a full navigation widget with lane guidance, speed limits, rerouting, and polished UI. Closest to Google Maps / Waze experience.

Pros

  • Full-featured out of the box
  • Lane guidance & speed limits
  • Automatic rerouting

Cons

  • Requires Navigation SDK license
  • Flood avoidance needs re-integration
  • Less UI customization
